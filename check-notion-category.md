# 카테고리 감지 문제 진단

## 현재 상황
- 총 100개 후기 모두 "bootcamp"로 분류됨
- 1대1컨설팅 후기가 0개

## 확인 필요사항

### GitHub Actions 로그에서 확인:

1. **"후기 #1 처리 중"** 같은 메시지 찾기
2. **"카테고리 필드 발견"** 메시지가 있는지 확인

예상되는 로그:
```
후기 #1 처리 중...
  → 카테고리 필드 발견: 프로그램 = 1대1컨설팅
  → 최종 카테고리: consulting
```

만약 "카테고리 필드 발견" 메시지가 **없다면**:
- "프로그램" 필드가 감지되지 않고 있음
- 필드 이름이 정확히 "프로그램"이 맞는지 확인 필요

만약 "카테고리 필드 발견" 메시지가 **있지만** 값이 다르다면:
- 실제 Select 옵션 값 확인 필요
- 예: "1:1 컨설팅", "커리어컨설팅" 등 다른 표기

## 요청사항

GitHub Actions 로그에서 **컨설팅 후기 1개의 전체 처리 과정**을 복사해주세요.

예:
```
후기 #X 처리 중...
  → 별점 필드 발견 (select): 별점
    [DEBUG] 원본 값: [5점]
    [OK] N점 파싱 성공: 5
  - 최종 별점: 5
  → 카테고리 필드 발견: ??? = ???  ← 이 부분!
  → 최종 카테고리: ???
✓ 후기 추가: XXX (5점, ???)
```
