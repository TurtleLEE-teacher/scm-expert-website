SCM(ERP) 초급반

기업 업무 프로세스 전체 그림 이해 + SCM 4대 Function 실무 체험

"전체적인 그림이 안 보여서 답답해요" 라는 고민을 해결하는 코멘토 직무부트캠프 수준의 기초 과정입니다.

기간: 5주 (3회 강의 + 4회 업무 체험)
대상: SCM 초보자, 취업 준비생, 직무 선택 고민자

주요 내용

  • ? 1주차: SCM 직무 개요 및 Planning Function
  • ? 2주차: Purchasing Function + SAP MM 기본
  • ? 3주차: Logistics Execution Function + SAP WM/LE
  • ? 4주차: Customer Service Function + SAP SD
  • ? 5주차: SCM 4개 Function 통합 및 커리어 패스 설계

SCM(ERP) 심화반

실제 기업에서 운영되는 복잡하고 심화된 SCM 프로세스 마스터

유상사급, 무상사급, 법인간거래, 외주공정생산 등 실제 기업의 복잡한 프로세스를 마스터합니다.

기간: 8주 (심화 SCM 프로세스 + 실무 프로젝트)
대상: 초급반 수료자, SCM 경험자, 복잡한 프로세스 이해 필요자

주요 내용

  • ? 1-2주차: 고급 SCM 프로세스 - 유상사급, 무상사급, 법인간거래
  • ? 3-4주차: 외주공정생산, 위탁가공, Subcontracting
  • ? 5-6주차: 산업군별 특화 프로세스 (JIT, 칸반, VMI, 3PL)
  • ? 7-8주차: 통합 프로젝트 및 프로세스 최적화
  • ? 실제 프로젝트 경험 기반 복잡한 비즈니스 룰 및 예외 처리
